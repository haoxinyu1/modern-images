API使用说明

您可以通过API上传图片，方便与第三方工具集成。

PicGo设置

在PicGo中添加自定义图床：

  • 上传地址:
  • 请求方式: POST
  • 自定义请求头: x-api-token 填入您的令牌
  • 图片上传字段名: images
  • 自定义URL部分: picgo=true
  • 返回值字段: result

API端点

  • /api/upload - 上传图片
  • 可选参数 format:
    • original - 保持原始格式
    • webp - 转换为WebP格式
    • avif - 转换为AVIF格式

认证方式

使用以下任一方式提供令牌：

  • 请求头: x-api-token: 您的令牌
  • URL参数: ?token=您的令牌
