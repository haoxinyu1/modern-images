 轻量图像存储解决方案，支持最新 **WebP / AVIF** 格式，内置 **格式自动转换** 与 **API接口上传** 功能。
 基于 **Node.js 高性能架构**，部署简单，使用便捷，适用于个人项目、博客、自建图床、前端协作等场景。
